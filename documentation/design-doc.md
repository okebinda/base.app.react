# Base React Native Mobile App
## Design Doc

### Introduction

This project is a starter package for a mobile app written in JavaScript with React Native.

### Technology Stack

* JavaScript
* React Native
* React Navigation
* Redux
* Axios
* i18next
* Immutable
* React Native Elements

### Features

* Login
* Forgot password
* Register
* Drawer navigation
* Tab navigation
* User account settings form
* User password update form
* Logout
* Temp screens
